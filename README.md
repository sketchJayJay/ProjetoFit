# EvoFit Pro

Projeto MVP em React + Vite para aplicativo de personal trainer.

## O que já tem
- Dashboard do personal
- Cadastro simples de alunos
- Área de treinos
- Evolução física
- Agenda
- Financeiro
- Simulação da área do aluno

## Como rodar
```bash
npm install
npm run dev
```
